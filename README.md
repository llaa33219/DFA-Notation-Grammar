# DFA-Notation-Grammar
선언성 자유 형식 표기 행동 문법 / Declarative Free-Form Action Notation Grammar
## 이것은 무엇인가?
이것은 ai와의 대화에 **행동**을 부여하기 위해 설계된 문법이다.
```
사용자: 나는 1+1이 3이라고 생각하는데 넌 어때?
AI: 저는 1+1은 2라고 사회적으로 약속되어 있기 때문에 2라고 생각합니다!
```
보다는
```
사용자: 나는 1+1이 3이라고 생각하는데 넌 어때?
AI: [{thinking}곰공히 생각하며][{move}사용자에게 다가간다]저는 1+1은 2라고 사회적으로 약속되어 있기 때문에 2라고 생각합니다!
```
이 더 감정적으로나 현실적으로나 더욱 친밀감이 느껴지고 상황 이해에 도움이 되기 때문에 이를 다양한 분야에서 깔끔하게 이용하기 위하여 만들어졌다.
